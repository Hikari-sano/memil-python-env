## WinPython setup

This environment uses WinPython as the only base Python runtime.

For this project, use this download page:

```text
https://sourceforge.net/projects/winpython/files/WinPython_3.12/3.12.10.1/
```

Recommended file:

```text
Winpython64-3.12.10.1dot.exe
```

ZIP version is also OK:

```text
Winpython64-3.12.10.1dot.zip
```

Beginner steps:

```text
1. Double-click WINPYTHON_SETUP.bat
2. Choose 1 to open the recommended download page
3. Download Winpython64-3.12.10.1dot.exe
4. Extract WinPython into the winpython folder
5. Run WINPYTHON_SETUP.bat again
6. Choose 3 to check placement
7. Run Start.bat
8. Run AI_CATALOG.bat
```

Expected layout:

```text
winpython/
└─ WPy64-xxxx/
   └─ python/
      └─ python.exe
```
