## WinPython setup made easy

This environment uses WinPython as the only base Python runtime.

For beginners:

```text
1. Double-click WINPYTHON_SETUP.bat
2. Download WinPython from the page that opens
3. Extract WinPython into the winpython folder
4. Run WINPYTHON_SETUP.bat again to check it
5. Run Start.bat
6. Run AI_CATALOG.bat
```

Expected layout:

```text
winpython/
└─ WPy64-xxxx/
   └─ python/
      └─ python.exe
```

If WinPython is not detected, run:

```text
WINPYTHON_SETUP.bat
```

or read:

```text
docs/WINPYTHON_BEGINNER_GUIDE.md
```
