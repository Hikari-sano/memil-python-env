# 設計メモ

## コンセプト

AviUtl2カタログのように、Python開発環境の導入・更新・テンプレート作成をまとめて扱う。

## 方針

- VS CodeはZIP版を使い、`vscode/data` に設定と拡張機能を保存する。
- Python本体と仮想環境管理はuvに任せる。
- `projects` 配下にユーザーのプロジェクトを置く。
- `catalog/catalog.json` を拡張して、テンプレート・拡張機能・Pythonパッケージを管理する。
