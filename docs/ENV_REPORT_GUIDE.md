# 環境レポートとファイル構造をAIに共有する方法

## 目的

研究者や学生が Python・VS Code・AIモデル環境でエラーが出たときに、
現在の環境情報とファイル構造をAIや担当者へ共有しやすくするための機能です。

## 使い方

1. `SHARE_ENV_TO_AI.bat` をダブルクリックします。
2. `env_reports` フォルダが開きます。
3. 以下の3種類のファイルが作成されます。

- `AI相談用_prompt_日時.txt`
- `env_report_日時.txt`
- `file_tree_日時.txt`

4. `AI相談用_prompt_日時.txt` を開き、相談内容を書き足します。
5. AIにその内容を貼り付けます。
6. 必要なら `env_report_日時.txt` と `file_tree_日時.txt` も共有します。

## file_tree に含まれるもの

- プロジェクト全体のフォルダ構造
- ファイル名
- ファイルサイズ

## file_tree に含まれないもの

- ファイルの中身
- `.git/`
- `vscode/`
- `python/`
- `cache/`
- `env_reports/`
- `.venv/`
- `runs/`
- 大きなモデルファイルやメディアファイル

## 注意

ファイル構造には、研究テーマや個人情報につながるファイル名が含まれる可能性があります。
外部へ共有する前に内容を確認してください。

`env_reports/` は `.gitignore` に追加して、GitHubへ上げない運用にしてください。
